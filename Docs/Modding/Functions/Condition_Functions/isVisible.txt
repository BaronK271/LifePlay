Actor.isVisible(Slot) returns bool

Check if an actor is wearing any clothes in a certain 'slot'. Will return false if it's an underwear hidden under other clothes. The slots are:
Outerwear
Headwear
Eyewear
Top
Top_Under
Bottom
Bottom_Under
Foot
Foot_Under

EXAMPLE:
    If John.isVisible(Bottom_Under)
        "John is wearing a pair of boxers and there's no other piece of clothing occupying the Bottom slot so these boxers are visible."
    Endif
